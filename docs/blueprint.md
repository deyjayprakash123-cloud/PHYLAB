# **App Name**: OUTR Physics Lab

## Core Features:

- Experiment Dashboard: Display all available experiments with a search option and dark/light mode toggle.
- Experiment Overview: Display aim, apparatus, theory, formulas, and diagrams for each experiment.
- Dynamic Observation Table: Allow users to input and edit observation data in a dynamic table with add/remove rows functionality.
- Auto Calculation Engine: Automatically perform calculations based on user input, including mean, standard deviation, and percentage error.
- Graph Generator: Generate labeled graphs with best-fit lines and slope values from user-inputted X and Y values; allow users to download the graph as a PNG.
- Report Generation: Export the lab report, complete with data, calculations, and graphs, as a PDF.
- Standard Values Reference: Provide a reference list of standard values (e.g., g, λ of sodium).

## Style Guidelines:

- Primary color: Use a calm blue (#64B5F6) to represent a scientific and reliable aesthetic.
- Background color: Light gray (#F0F0F0) for a clean and modern look, ensuring readability.
- Accent color: A muted orange (#FFAB40) to highlight key interactive elements without distracting from the data.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral, and objective feel suitable for both headlines and body text.
- Use clear, minimalistic icons to represent different experiments and functionalities.
- Implement a clean, modern, and responsive layout for easy navigation and accessibility on various devices.
- Subtle transitions and animations to enhance user experience without being distracting.
